# Sistema Web - NutriGym

## Descripción
Sistema web desarrollado con Laravel, MySQL, Tailwind CSS y Alpine.js.

##  Stack Tecnológico

=> Backend
-> Laravel 10 - Framework PHP
-> MySQL 8.0 - Base de datos relacional -> phpMyAdmin 

### Frontend
-> Tailwind CSS** - Framework CSS utility-first
-> Alpine.js** - Framework JavaScript para interactividad
-> Blade Templates** - Motor de plantillas de Laravel

## 📋 Prerrequisitos

- PHP 8.1 o superior
- Composer 2.0 o superior
- Node.js 16.0 o superior
- MySQL 8.0 o superior

Este README incluye solo la información esencial y las instrucciones básicas para configurar y usar el proyecto con el stack tecnológico especificado.
